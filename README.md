PALINDROME CHECKER TOOL

DESCRIPTION

This is a simple Python utility designed to check if a given word, phrase,
or sequence is a palindrome (reads the same backward as forward). The tool
is case-insensitive and handles user input via the command line.

FILES INCLUDED

palindromechecker.py  - The main Python source code.

submission.pdf         - The formal report (generated from submission.tex).

statement.md           - The original problem statement.

HOW TO RUN

Ensure you have Python 3 installed on your system.

Open your terminal (Mac/Linux) or Command Prompt (Windows).

Navigate to the folder containing 'palindrome_checker.py'.

Type the following command and press Enter:

python palindrome_checker.py

USAGE EXAMPLE

When the program starts, you will see a prompt. Enter a word to check.

Enter a word to check: Level
'level' IS a palindrome.

Enter a word to check: Hello
'hello' is NOT a palindrome.
